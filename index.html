Prova HTML no laboratorio

HTML é a linguagem padrão para criar páginas da web. É uma linguagem de marcação que utiliza várias tags para estruturar o conteúdo de uma página da web, como texto, imagens, vídeos e links para outros documentos. !DOCTYPE html: Define o tipo de documento HTML. html: A tag raiz que envolve todo o conteúdo da página. title: A tag que define o título da página, que é exibido na aba do navegador. Para exibir uma imagem em uma página HTML, você pode usar a tag e especificar o caminho para a imagem no atributo src. A tag head contém informações sobre o documento, como o título da página, referências a arquivos externos (como folhas de estilo e scripts) e meta informações. A tag body contém o conteúdo principal da página. As tags HTML são elementos usados para marcar diferentes partes do conteúdo. Algumas tags comuns incluem h1 a h6 para títulos, p para parágrafos, a para links, img para imagens e assim por diante. As tags podem ter atributos que fornecem informações adicionais sobre a tag. Por exemplo, a tag a pode ter o atributo href para especificar o URL de destino do link. O HTML também permite criar formulários usando a tag form e vários tipos de elementos de entrada, como input, select, textarea, entre outros. O HTML fornece a estrutura básica para a construção de uma página da web, mas é frequentemente combinado com CSS (Cascading Style Sheets) e JavaScript para melhorar o design e a funcionalidade da página.
